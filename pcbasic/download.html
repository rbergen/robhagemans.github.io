PC-BASIC will run on any system that supports Python. The packaged releases for Windows, OS X and Linux run out of the box on x86 platforms; for other platforms and operating systems, just download the source distribution from the table above and install the required Python packages. See the PC-BASIC README for a list of packages and installation instructions.

This and older releases can also be found on the PC-BASIC release page on GitHub.

PC-BASIC is still in beta; there is as of yet no stable release available. However, the development releases are generally very usable. Should you find a bug or wish to request a feature, please post a message on the discussion page.
