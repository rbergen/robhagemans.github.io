PC-BASIC release page on GitHub Linux, FreeBSD or any other systems supporting Python 2.7

PC-BASIC will run on any system that supports Python. The packaged releases for Windows, OS X run out of the box. So does the source release on Debian, Ubuntu, Mint and Fedora on all architectures. On other Linux and Unix systems, the source distribution will install automatically but you will need to manually install dependencies. See the PC-BASIC README for a list of packages and installation instructions.

This and older releases can also be found on the PC-BASIC release page on GitHub.

PC-BASIC is still in beta; there is as of yet no stable release available. However, the development releases are generally very usable. Should you find a bug or wish to request a feature, please post a message on the discussion page.
